# Chrome Leads Tracker Extension



A simple Chrome extension built with vanilla JavaScript that allows users to save URLs for later reference.



Users can:

- Save a custom URL via text input

- Save the currently active browser tab

- Persist saved links using `localStorage`

- Open saved links in new tabs

- Clear all saved links with a double-click



---



## Features

- Save manual input URLs

- Save the active Chrome tab URL

- Persistent storage using `localStorage`

- Clickable saved links

- Clear all saved data



---



## Built With

- HTML

- CSS

- JavaScript

- Chrome Extensions API (`chrome.tabs`)



---



## How to Run Locally



1. Open Chrome and navigate to: chrome://extensions
2. Enable **Developer mode** (top-right)
3. Click **Load unpacked**
4. Select this project folder
5. The extension will appear in your toolbar
